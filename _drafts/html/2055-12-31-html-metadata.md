## Вводная 

В статье [https://lexusalex.site/posts/html-base-tags/](https://lexusalex.site/posts/html-base-tags/) мы рассмотрели базовые теги среди которых был тег `<head>`. 

Сегодня рассмотрим что такое метаданные страницы.

Метаданные страницы - это данные о данных, которые не отображаются непосредственно на самой странице, но предоставляют информацию о ней для браузеров, поисковых систем и других веб-сервисов. 

Они содержатся в элементе `<head>`.

Элементы также могут предоставлять внешнюю информацию для страницы.

## base

`<base>` предназначен для определения базового `url` относительно всех адресов на странице.

````html
<base href="http://site.com/">
````

### Назначение

Элемент `<base>` определяет базовый `url` относительно которого будут работать ссылки, скрипты, изображения на странице.
Так же `<base>` указывает на целевое окно или фрейм где будут открыты ссылки.

### Особенности

- `<base>` - это одинарный тег
- Элемент `<base>` должен располагаться внутри `<head>`, и на странице он должен быть один.
- `<base>` может подойти если на сайте используется сложная структура каталогов.
- В большинстве случаев нет необходимости использовать `<base>`, так как использование абсолютных адресов является более простым и понятным решением.
- Если `<base>` все же используется, нужно расположить его как можно выше к в теге `<head>`.

### Родительские/Дочерние элементы

В качестве родителя может быть только `<head>`, а в дочерних ничего быть не может, так как тег одиночный

### Атрибуты

#### href

Определяет основной адрес страницы, то есть базовый `URL` для всех относительных `URL`-адресов в документе.

````html
<base href="http:/test-site-one.com">
````

#### target

Указывает где будут открываться ссылки и формы, то есть целевое окно или фрейм для открытия ссылок в документе.

##### _self

Открывает ссылку в текущем окне или фрейме (по умолчанию).

````html
<base href="http:/test-site-one.com" target="_self">
````

##### _parent

Открывает ссылку в родительском фрейме.

````html
<base href="http:/test-site-one.com" target="_parent">
````

##### _top

Открывает ссылку во всем окне.

````html
<base href="http:/test-site-one.com" target="_top">
````

##### _blank

Ссылка будет открыта в новой вкладке.

````html
<base href="http:/test-site-one.com" target="_blank">
````

> Подробнее механизм открытия ссылок мы разберем когда будет рассматривать элемент iframe
{: .prompt-info }

Так же можно указать имя фрейма, где будет результат ссылки.

Элемент поддерживает глобальные атрибуты и события, но так как это технический элемент, они не нужны.

### Примеры

#### Пример разметки

````html
<!DOCTYPE html>
<html lang="en">
  <head>
      <meta charset="UTF-8">
      <title>Title</title>
      <base href="http:/test-site-one.com">
  </head>
  <body>
  
  <a href="/super-puper-url.html">ссылка</a>
  <img src="/photos/northern-lights-over-snow-capped-mountian-LtnPejWDSAY">
  
  </body>
</html>
````

### Поддержка браузерами

Поддерживается всеми основными браузерами

### Стили по умолчанию

````css
base {
   display: none;
}
````
