### useEffect

Специальный хук `useEffect` может помочь если нужно загрузить внешние данные, создание интервала и т.д.

````jsx
useEffect(() => {
  console.log(new Date().toLocaleTimeString('ru-RU'));
}, []);
````


````jsx
useEffect(() => {
  setInterval( () => {
    setTime(new Date().toLocaleTimeString('ru-RU'));
  },1000)
}, [1]);
````
