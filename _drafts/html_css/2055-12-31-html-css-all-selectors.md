## Базовая таблица селекторов

| Селектор    |       Назначение       |                     Описание                     |          Что попадает          | Дополнительно       | Поддержка  |
|-------------|:----------------------:|:------------------------------------------------:|:------------------------------:|---------------------|------------|
| `*`         | Универсальный селектор |        Попадают все элементы на странице         |   `<body><div></div></body>`   |                     | `baseline` |
| `div`       |     Селектор тега      |          Все элементы `div` на странице          |         `<div></div>`          |                     | `baseline` |
| `#header`   |    Селектор по `id`    |             Элемент с указанным `id`             |   `<div id="header"></div>`    |                     | `baseline` |
| `.title`    |   Селектор по классу   |         Все элементы с указанным классом         | `<span class='title'></span>`  |                     | `baseline` |
| `.red.blue` |   Составной селектор   | Применим для классов и атрибутов, пишется слитно | `<div class='red blue'></div>` | #Составной селектор | `baseline` |
