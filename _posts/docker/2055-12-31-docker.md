## С какими проблемами сталкиваемся

- Конфликт версий
- На одном сервере работает, на другом нет
- Не возможно масштабирование
- Конфликт портов

## Что дает docker

- Изолированное окружение для запуска
- Простое управление приложением
- Легкое масштабирование
